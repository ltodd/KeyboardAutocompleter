# KeyboardAutocompleter
Autocompetes Keyboard

KeyboardAutocompleter contains implementations (really just one right now) of the Autocompleter interface.

# WordCountAutocompleter
	Suggests words based soley on their word count in training passages
	
	If you'd like to try WordCountAutocompleter, in the project root, run:
	
	mvn package
	
	java -cp target/KeyboardAutocompleter-0.0.1-SNAPSHOT.jar Autocompleter.WordCountAutocompleter
	
	This will give you a very basic interface to play with the module.
